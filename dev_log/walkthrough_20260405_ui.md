# WHTOOLS Dashboard UI Polish: Enhanced About Dialog (v5.9.3)

안녕하세요, **WHTOOLS**입니다.

기존에 간소화되었던 'About' 상자를 다시 풍성하고 전문적인 디자인으로 복구했습니다. 이제 WHTOOLS의 핵심 기술력과 브랜드 이미지가 사용자에게 명확히 전달됩니다.

## 🛠️ 주요 개선 사항

### 1. 프리미엄 'About' 다이얼로그 구현
- **브랜드 로고 센터링**: `resources/logo.png` 이미지를 상단 중앙에 배치하여 시각적 완성도를 높였습니다.
- **핵심 기술 요약**: Kirchhoff/Mindlin/Von Karman 이론, JAX SSR 엔진, Digital Twin 동기화 등 대시보드의 주요 기능을 불렛 포인트로 정리하여 전문가용 도구로서의 가치를 강조했습니다.
- **세련된 UI 구성**: 제목, 버전 정보, 저작권 문구 등을 조화로운 폰트 크기와 색상으로 배치했습니다.

### 2. 리소스 경로 정규화
- **경로 수정**: 로고 이미지와 `D2Coding` 폰트가 위치한 `resources` 하위 디렉토리를 정확히 참조하도록 내부 경로 로직을 수정했습니다.
- **안정성 확보**: 리소스 파일 존재 여부를 체크하여 폰트 미설치 시에도 시스템 기본 폰트로 안전하게 폴백되도록 구성했습니다.

## 🧪 실행 확인

> [!TIP]
> **Help > About** 메뉴를 클릭하여 새롭게 바뀐 디자인을 확인해 보세요. WHTOOLS의 정체성이 담긴 깔끔한 기능을 만나실 수 있습니다.
