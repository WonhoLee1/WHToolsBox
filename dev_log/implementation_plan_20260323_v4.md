# [구현 계획서] 완충재 영구 변형(Plasticity) 기하 로직 정밀화 (2026-03-23)

안녕하세요, **PROCPA**입니다.

완충재 블록의 영구 변형 로직에서 고정 비율(8:2) 대신, 블록의 초기 크기와 물리적 거동을 고려한 기하학적 정밀 계산 방식을 적용하겠습니다.

## 1. 개요
현재의 영구 변형 로직은 경험적인 비율(이동 80%, 축소 20%)을 사용하여 시각적 효과에 치중해 있었습니다. 이를 블록의 전체 변형량을 기준으로 하여, 한쪽 면이 고정된 채 반대편 면만 압축되는 물리적으로 보다 타당한 기하 모델(이동 50%, 축소 50%)로 개선합니다.

## 2. 변경 사항
- **`run_drop_simulation.py` 내 `apply_plastic_deformation` 함수 수정**:
    - `inward_dir` 계산 로직 수정: 개별 블록 바디의 중심점이 아닌, 전체 어셈블리(BPackagingBox)의 중심 방향으로의 벡터를 활용하여 정확한 압축 축을 식별합니다.
    - **변형량 분배**: 전체 변형량(`deformation`)의 절반은 크기 축소(`half_shrink`)에, 나머지 절반은 위치 이동(`shift_amount`)에 할당하여 상단 면의 위치를 유지합니다.
    - **축 특정 변형**: 공기 저항 및 인접 블록 간섭을 최소화하기 위해 주 압축 축(Major axis)에 대해서만 변형을 적용합니다.
