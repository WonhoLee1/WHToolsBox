# WHTOOLS Dashboard UI Polish: Mesh Edge Visibility Restoration (v5.9.6)

안녕하세요, **WHTOOLS**입니다.

3D 뷰의 컨텍스트 메뉴(우클릭 메뉴)에서 유실되었던 **'Show Mesh Edges'** 기능을 성공적으로 복구했습니다.

## 🛠️ 주요 변경 사항

### 1. 컨텍스트 메뉴 Mesh Edge 토글 추가
- **기능 복구**: 3D 뷰 위에서 마우스 우클릭 시 나타나는 메뉴에 **'Show Mesh Edges'** 항목을 다시 추가했습니다.
- **실시간 반응**: 체크박스 형태로 현재 상태(On/Off)가 표시되며, 변경 시 모든 파트의 메쉬 라인(Edges)이 즉각적으로 표시되거나 숨겨집니다.
- **사용자 편의성**: 복잡한 변형 형상을 확인할 때는 메쉬 라인을 숨겨 컨투어 가시성을 높이고, 정확한 절점 위치가 필요할 때는 다시 켤 수 있어 분석 작업의 유연성이 높아졌습니다.

## 🧪 실행 확인

> [!TIP]
