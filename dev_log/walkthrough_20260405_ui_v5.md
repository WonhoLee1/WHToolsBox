# WHTOOLS Dashboard UI Polish: Mesh Edge Color Update (v5.9.7)

안녕하세요, **WHTOOLS**입니다.

해석 결과 확인 시 가시성을 극대화하기 위해 메쉬 라인(Mesh Edge)의 색상을 **'darkgray'**로 업데이트했습니다.

## 🛠️ 주요 변경 사항

### 1. 메쉬 라인 색상 심화 ('lightgrey' → 'darkgray')
- **시인성 개선**: 기존의 연한 회색(`lightgrey`)은 흰색 배경이나 밝은 컨투어 영역에서 다소 흐릿하게 보이던 점을 개선했습니다.
- **대조 효과**: 이제 **'darkgray'** 색상이 적용되어 파트의 격자 구조가 더욱 뚜렷하게 관찰되며, 변형 전후의 메쉬 왜곡 상태를 직관적으로 파악할 수 있습니다.
