# MuJoCo 재질 파라미터(solref/solimp) 최적화 및 물리적 고찰 리포트 (보완본)
**작성일**: 2026-03-22
**대상**: MuJoCo 반발계수(COR) 최적화 로직, 헤르츠 접촉 이론 기반 강성 산출 및 수치 안정성 분석

---

## 1. 개요 (Background)
본 시뮬레이션 시스템은 실제 물리적 재질의 탄성계수($E$)와 밀도($\rho$)를 기반으로 MuJoCo의 `solref` 파라미터를 자동 최적화하는 것을 목표로 함. 특히, 헤르츠 접촉 이론(Hertzian Contact Theory)을 통해 물리적 강성을 수치적 가속도 파라미터로 변환하는 과정을 정밀하게 모델링함.

---

## 2. 물리 모델링 구현 상세 (Mathematical Foundations)

### 2.1. 헤르츠 접촉 기반 강성 ($K$) 도출 (음수 방식)
헤르츠 이론에 따르면 구(Sphere)와 평면간의 비선형 접촉력($F$) 및 침투 깊이($\delta$)의 관계는 다음과 같음:
$$F = \frac{4}{3} E^* \sqrt{R} \delta^{1.5}$$
*   $E^*$: 유효 탄성 계수 ($E / (1 - \nu^2)$)
*   $R$: 구의 반경
*   $\delta$: 침투 깊이 (Penetration)

강성($K$)은 힘의 변화율($dF/d\delta$)이므로:
$$K(\delta) = 2 E^* \sqrt{R \delta}$$

**[선형화 로직]**
MuJoCo의 음수 `solref`($p1, p2$)는 선형 스프링-댐퍼 모델을 전제하므로, 특정 침투 지점에서 이 비선형 곡선을 선형화함. 본 시스템은 침투 깊이가 구 반경의 1%인 지점($\delta = 0.01R$)을 기준 강성 산출 포인트로 선정함.
*   **산출 공식**: $K_{raw} = 2 E^* \sqrt{R \cdot (0.01R)} = 0.2 E^* R$

### 2.2. 접촉 지속 시간 ($T_c$) 기반 시상수 산출 (양수 방식)
양수 방식의 `timeconst`($d$)를 결정하기 위해 헤르츠 이론에 기반한 충돌 지속 시간 공식을 사용함:
$$T_c = 2.9432 \left( \frac{m^2}{R E^{*2} v_{impact}} \right)^{1/5}$$
*   $v_{impact}$: 충돌 직전 수직 속도 ($\sqrt{2gh}$)
*   $m$: 물체의 질량

이 $T_c$는 물체가 바닥에 머무는 전체 시간을 의미하며, 이를 `timeconst`로 입력하여 MuJoCo 엔진이 재질별 고유 진동수를 모사하도록 함.

## 3. 파라미터의 물리적 해석 및 한계

### 3.1. 음수 방식 파라미터 ($p1, p2$) 의 본질
- **공식**: $p1 = -K / m$, $p2 = -B / m$
- **특성**: 이 파라미터들은 가속도 평면에서의 계수임. 충돌 가속도($a$)는 $a = p1 \cdot x + p2 \cdot u$로 결정되며, 여기에는 질량($m$) 항이 존재하지 않음.
- **물리적 한계 (Mass Independence)**: 질량이 바뀌어도 동일한 침투/속도 조건이라면 가속도 궤적이 동일하게 유지됨. 즉, 질량에 따라 반발력이 스케일링되어 버리므로, 질량 변화에 따른 동적 응답의 변화(침투 깊이 증가 등)를 본 모델로는 정확히 설명하거나 예측할 수 없음. (Acceleration-locked 모델)

### 3.2. 양수 방식의 불안정성 원인 분석 (Numerical Instability)
동일한 타임스텝($dt$) 하에서 음수 방식은 안정적이나 양수 방식이 불안정한 이유는 다음과 같음:
- **수치적 예민도**: 양수 방식($d, r$)은 내부적으로 강성($K$)을 $1/d^2$의 관계로 변환함. $d$가 작아질수록(단단한 재질) 강성이 제곱비로 급증하며, 타임스텝 경계에서 수치 에너지가 폭주(Divergence)하기 쉬움.
- **경고 현황**: 현재 양수 방식은 낮은 COR 탐색 시 `Nan` 오류 및 `mixed solref` 경고가 잦으며, 이는 $solimp$ 파라미터와의 선형 결합 과정에서 수치적 덴핑이 음수 형태보다 훨씬 불안정하게 작용하기 때문임. (향후 개선 필요)

## 4. 결론 및 향후 개선 과제
1.  **Stiffness Scaling**: 현재 시뮬레이션 안정성을 위해 $E$를 10~100배 낮추어 사용 중임. 타임스텝과 맞물린 물리적 한계를 극복하기 위한 수치 적분기 튜닝 검토 필요.
2.  **Model Validity**: 음수 방식의 질량 독립성 문제를 인식하고, 가중치 최적화 시 이를 보정할 수 있는 질량 비례 항 추가 연구 요구됨.
3.  **양수 방식 고도화**: 기본 세팅인 양수 방식의 안정성 확보를 위해, $d$와 $dt$ 사이의 최소 비(Safety margin)를 자동으로 튜닝하는 알고리즘 도입 예정.
