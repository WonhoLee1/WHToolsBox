# WHTOOLS Dashboard UI Polish: Toolbar Height Stabilization (v5.9.8)

안녕하세요, **WHTOOLS**입니다.

애니메이션 컨트롤 툴바(`Animation Control Toolbar`)의 높이가 폰트나 버튼의 영향을 받지 않도록 **고정 높이(Fixed Height)**를 적용하여 레이아웃을 안정화했습니다.

## 🛠️ 주요 변경 사항

### 1. 툴바 높이 고정 (40px)
- **레이아웃 안정성**: 툴바 자체의 높이를 **40px**로 고정하여, 내부 위젯의 미세한 크기 변화가 상단 레이아웃 전체를 흔들지 않도록 조치했습니다.
- **위젯 높이 통일**: 툴바 내부에 포함된 재생 버튼, 프레임 레이블, 드롭다운 박스(Step, Speed) 등의 높이를 모두 **26px**로 통일하여 완벽한 수평 정렬을 구현했습니다.

### 2. 로고 크기 동기화 (100px)
- **픽셀 퍼펙트**: 사용자님의 요청에 따라 로고 위젯의 높이를 **100px**로 고정함과 동시에, 이미지 스케일링 엔진(`scaledToHeight`) 역시 **100px**로 일치시켜 잘림 현상 없는 깔끔한 출력 상태를 복구했습니다.

## 🧪 실행 확인

> [!CHECK]
> 이제 폰트 설정을 변경하거나 버튼을 클릭해도 툴바의 높이가 고정되어 있어, 화면 깜빡임이나 레이아웃 밀림 현상 없이 쾌적하게 사용하실 수 있습니다.
