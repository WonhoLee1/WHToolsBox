# Walkthrough: Post-Processing UI v4.2 Improvements

사용자의 피드백을 반영하여 **모션 분석 (Motion Analysis)** 기능을 강화하고 UI 편의성을 대폭 개선하였습니다.

## 1. 주요 개선 사항 (Key Features)

### 1.1. 모션 분석(Motion Analysis) 명칭 정립
- 기존 '기구학 분석 (Kinematics)' 탭과 관련 메시지를 모두 **'모션 분석 (Motion Analysis)'**으로 변경하였습니다. 
- 이는 공학적으로 물체의 전체적인 움직임에 집중하는 분석임을 명확히 합니다.

### 1.2. 좌표계 선택 기능 (Global/Local)
- **Global (절대)**: MuJoCo 월드 좌표계를 기준으로 한 위치값을 보여줍니다 (기본값). 낙하 높이(0.5m) 등이 그대로 그래프에 반영됩니다.
- **Local (상대)**: 시작 시점($t=0$)을 0으로 하는 변위값을 보여줍니다. 충격 시점의 변위량만 확인하고 싶을 때 유용합니다.

### 1.3. 그래프 품질 및 테마 제어
- **축 레이블 추가**: X축(`Time (s)`), Y축(`Pos (m)`, `Vel (m/s)`, `Acc (m/s^2)`) 레이블을 모든 그래프 모드에 추가하였습니다.
- **주제(Theme) 변경**: 상단 메뉴 `그래프 옵션` -> `그래프 테마 (Theme)`를 통해 `ggplot`, `seaborn`, `dark_background` 등 Matplotlib 테마를 실시간으로 변경 가능합니다.

### 1.4. 편의성 강화 (Select/Deselect All)
- **1. 데이터 종류** 및 **3. 추출 위치** 섹션에 각각 `전체 선택`, `전체 해제` 버튼을 추가하여 다수의 코너나 데이터를 한 번에 편리하게 제어할 수 있습니다.

### 1.5. 백엔드 시스템 최적화
- **PySide 지원**: 기본 백엔드를 `QtAgg`로 시도하도록 설정하여, PySide 기반의 고성능 대화형 그래프 창을 사용할 수 있도록 하였습니다.
- **메뉴 정리**: 불필요한 백엔드 전환 메뉴를 삭제하여 인터페이스를 간소화했습니다.

## 2. 사용 방법
1. **모션 분석** 탭을 선택합니다.
2. `좌표계 선택`에서 **Global**이 선택되어 있는지 확인합니다.
3. `전체 선택` 버튼을 눌러 원하는 코너들을 한 번에 체크합니다.
4. `매트릭스 그래프 생성`을 눌러 단위가 포함된 절대 좌표 그래프를 확인합니다.
5. 상단 메뉴에서 테마를 변경하여 원하는 스타일의 결과를 얻습니다.
