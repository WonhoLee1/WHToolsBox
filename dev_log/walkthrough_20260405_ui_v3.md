# WHTOOLS Dashboard UI Polish: About Box Logo Resize (v5.9.5)

안녕하세요, **WHTOOLS**입니다.

사용자 정보 창('About' 창)의 시각적 임팩트를 강화하기 위해 로고 크기를 대폭 확대했습니다.

## 🛠️ 주요 변경 사항

### 1. 로고 크기 2배 확대 (240px)
- **비주얼 강화**: `About` 창 상단의 WHTOOLS 로고를 기존 120px에서 **240px로 2배 확대**하여 브랜드 정체성을 더욱 뚜렷하게 강조했습니다.
- **고해상도 유지**: `SmoothTransformation` 옵션을 적용하여 확대 시에도 깔끔하고 선명한 화질을 유지합니다.

### 2. 다이얼로그 레이아웃 최적화
- **창 크기 조정**: 커진 로고가 다른 기술 정보들과 겹치지 않도록 창의 높이를 **480px에서 600px로 확장**했습니다.
- **균형 잡힌 배치**: 로고, 버전 정보, 기술 스택 리스트 간의 여백을 재조정하여 전문적인 느낌의 레이아웃을 완성했습니다.

## 🧪 실행 확인

> [!TIP]
> **Help > About** 메뉴를 다시 확인해 보세요. 더욱 압도적인 존재감의 로고와 정돈된 안내 문구를 확인하실 수 있습니다.
