# Walkthrough - Plate Assembly & Advanced Alignment (v2)

`plate_by_markers_v2.py` 구현을 완료하였습니다. 본 버전은 단일 판 해석에서 벗어나 여러 파트가 조립된 **Assembly** 단위의 구조 해석 및 시각화를 지원합니다.

## 1. 주요 변경 사항 및 신규 기능

### 1.1. 다중 파트 관리 (Assembly Manager)
- `PlateAssemblyManager`를 통해 여러 개의 판(Plate)을 하나의 시스템으로 관리합니다.
- 각 파트는 독립적인 물성, 규격, 마커 데이터를 가지며 JAX를 통해 일괄적으로 고속 해석됩니다.

### 1.2. 3D 뷰포트 및 상단 배너 복구
- **배너 로고**: 상단 좌측에 `WHTOOLS` 공식 배너 로고가 정상적으로 복구되었습니다.
- **우클릭 메뉴**: 3D 뷰포트에서 마우스 오른쪽 버튼을 클릭하여 `Parts Visibility` 메뉴를 호출, 개별 면(Face)의 가시성을 조절할 수 있습니다.

### 1.3. 2D 그래프 레이아웃 최적화 (Part-Key 매칭)
- 2D 그래프 컨트롤 영역을 2x2 그리드로 재배치하였습니다.
- 각 차트(F1, F2, C1, C2)마다 독립적인 **`Part`** 선택 드롭다운과 필드/히스토리 선택 드롭다운이 1:1로 매칭되어 있어, 어떤 부품의 어떤 데이터를 보고 있는지 직관적으로 파악할 수 있습니다.

### 1.4. 규격 및 오프셋 기반 마커 정렬 (Alignment Logic)
- 사용자가 입력한 판의 `W`, `H` 규격과 마커의 `Offsets`를 기준으로 최적의 로컬 좌표계를 추정합니다.
- Procrustes 분석 알고리즘을 적용하여, 실측 마커 데이터가 규격 상의 어느 위치에 해당하는지를 자동으로 매칭하여 해석의 신뢰도를 높였습니다.

## 2. 고도화된 검증 시나리오: 현실적 포장 상자 (Packaging Box)

기존의 단순 정육면체 모델을 **2000 x 1200 x 800 mm** 규격의 실제 포장용 박스 모델로 업그레이드하였습니다.

- **물리적 거동**: 자유낙하 후 지면에 충격하면서 발생하는 튀어오름(Bouncing)과 마찰에 의한 복합 전복(Tumbling) 거동을 정교하게 모사합니다.
- **충돌 변형 모사**: 박스의 각 면이 지면에 근접하거나 접촉할 때만 변형(`Ripple`)이 강하게 발생하도록 가중치 알고리즘을 적용하여, 실제 낙하 테스트 시 데이터와 유사한 경향성을 갖도록 하였습니다.

## 3. 사용 가이드

1. **실행**: `python plate_by_markers_v2.py` 명령으로 대시보드를 엽니다.
2. **파트 제어**: 3D 화면 우클릭을 통해 보고 싶은 면만 활성화합니다.
3. **데이터 비교**: 2D Plot 영역 상단의 `Part` 선택기를 조절하여 분석할 파트를 지정합니다.
4. **범위 조정**: `Legend` 모드를 `Static`으로 변경하고 `Min/Max`를 조절하여 미세한 응력 변화를 관찰합니다.
