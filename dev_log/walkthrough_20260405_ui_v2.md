# WHTOOLS Dashboard UI Refinement (v5.9.4)

안녕하세요, **WHTOOLS**입니다.

사용자 인터페이스의 세밀한 완성도를 높이기 위해 'Add Plot' 다이얼로그와 메인 배너 레이아웃을 정제했습니다. 주요 변경 사항은 다음과 같습니다.

## 🛠️ 주요 개선 사항

### 1. 2D Plot 메뉴 전향적 개편 (Radio Buttons)
- **조작성 개선**: 기존 드롭다운(Combo Box) 방식의 'Type' 선택을 **라디오 버튼 (Contour / Curve)** 방식으로 변경했습니다.
- **상태 가시성**: 현재 선택된 플롯 타입을 한눈에 파악할 수 있으며, 클릭 한 번으로 타입 전환 및 키 리스트 업데이트가 즉각적으로 이루어집니다.

### 2. 브랜드 배너 크기 최적화 (+15%)
- **배너 리사이징**: 상단 배너의 로고 크기를 **100px에서 115px로 15% 확대**하여 브랜드 가시성을 확보했습니다.
- **레이아웃 유지**: 주변 위젯 및 3D View Control 그룹박스와의 조화를 유지하면서도 로고 본연의 임팩트를 강화했습니다.

### 3. 리소스 및 정보 창 복구 (이전 단계 포함)
- **About 창 강화**: WHTOOLS 로고와 핵심 기술(Shell Theory, JAX SSR 등)이 포함된 프리미엄 정보 창을 재구축했습니다.
- **리소스 경로 정규화**: `resources` 폴더 내의 로고와 폰트를 정확히 참조하도록 경로 매핑을 수정했습니다.
