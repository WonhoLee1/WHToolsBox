# [Goal] 3D-2D 분리형 전문가용 대시보드 구조 개편 (2026-04-05)

현재의 고정된 레이아웃을 탈피하여, 3D 분석과 2D 데이터 추출을 독립적으로 수행할 수 있는 스플리터(Splitter) 기반의 고도화된 UI 구조로 개편합니다. 특히 2D 플롯 영역은 유저가 실시간으로 하위 레이아웃을 정의하고 원하는 데이터를 할당할 수 있는 동적 엔진으로 재설계됩니다.

## Proposed Changes

- **Layout**: QSplitter 도입 (좌: 3D, 우: 2D)
- **Menu**: View 메뉴 (3D/2D 토글) 및 상태바 추가
- **Toolbar**: 상단 애니메이션 툴바 (Play, Slider, Step Combo)
- **2D Engine**: 1x1~3x2 동적 레이아웃 및 "Add Plot" 다이얼로그 시스템
- **Pop-out**: 현재 데이터 및 툴바 포함 새 창 생성

## Verification Plan

- 레이아웃 전환 및 리사이징 테스트
- 동적 플롯 할당 및 렌더링 검증
- 애니메이션 스텝 기능 검증
