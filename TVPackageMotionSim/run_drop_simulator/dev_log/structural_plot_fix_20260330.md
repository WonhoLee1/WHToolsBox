# [2026-03-30] 포스트 프로세싱 구조 해석 그래프 시각화 개선 (v4.3.2)

## 1. 개요

- RRG, Bending Stress(BS), Torsional Stress(TS) 등 미세 지표가 y축 범위 설정(0~1 고정)으로 인해 보이지 않는 문제 해결.
- 그래프 레이아웃을 1열 고정에서 그리드(2열) 방식으로 확장하여 가독성 향상.
- 피규어 및 서브플롯 크기를 최적화 (가로 2/3, 세로 2배).
- 긴 y축 제목이 잘리는 문제를 해결하기 위해 자동 줄바꿈 적용.

## 2. 세부 변경 사항

### 2.1. 그래프 레이아웃 및 크기 조정

- 선택된 지표가 3개 이상일 경우 2열(Grid) 레이아웃으로 변경.
- 지표별 서브플롯 크기 조정:
  - 개별 플롯 가로: 기존 10 -> 약 6.7 (2/3 수준)
  - 개별 플롯 세로: 기존 2.5 -> 5.0 (2배 수준)
- 전체 피규어 크기는 서브플롯 개수에 따라 동적 계산.

### 2.2. Y축 범위 최적화

- 기존 `ax.set_ylim(0, max(1.0, curr_ylim[1] * 1.1))` 코드가 작은 값(예: 0.1 MPa)을 가지는 BS, TS 등을 0 근처에 뭉개버리는 문제 수정.
- `ax.set_ylim(bottom=0)`만 설정하여 상단 데이터는 Matplotlib의 오토스케일링에 맡김.

### 2.3. 제목 및 레이블 개선

- Y축 제목이 길 경우(예: PBA Magnitude) 자동으로 줄바꿈(`\n`)을 삽입하여 피규어 외부로 나가지 않도록 함.
- 그리드 레이아웃 적용 시 겹침 방지를 위해 `tight_layout` 재조정.

## 3. 적용 방법
